Details about the second batch of problems will be announced on March 14, 2026.

Methodology for the Second Batch

For the second batch, we will implement a benchmarking phase prior to the community release. The benchmark phase will be designed to ensure the following features:

  • Verification that the solutions are produced autonomously.
  • A formal grading scheme and refereeing, modeled on the journal review system.
  • An explicit description of the problem selection process, including advance internal testing on systems which have a zero data retention policy.

If you are interested in an assessment of your solutions to the second round of questions, please email contact@1stproof.org.
